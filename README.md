## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- MongoDB
- Redis
- Telegram Bot Token (from @BotFather)
- Razorpay Account

### Installation

```bash
# Clone and install backend
cd backend
npm install
cp .env.example .env
# Edit .env with your credentials

# Start backend
npm run dev

# Install admin panel
cd ../admin
npm install
npm run dev
```
